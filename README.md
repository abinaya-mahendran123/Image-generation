# 🐼 Panda Image Generator using API

This is a simple Python mini project that generates an image using an online API based on a text prompt.

## 🚀 Project Description

This project uses the **Pollinations AI Image API** to generate an image from a given text prompt.
In this example, the prompt used is:

> *"panda with flower"*

The generated image is then saved locally as a `.png` file.

---

## 🛠️ Technologies Used

* Python 🐍
* `requests` library
* REST API (Pollinations AI)

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/panda-image-generator.git
cd panda-image-generator
```

2. Install required library:

```bash
pip install requests
```

---

## ▶️ Usage

Run the Python script:

```bash
python app.py
```

---


## 🖼️ Output

The generated image will be saved as:

```
pt.png
```

You can open it to view the AI-generated image.
